# TeleLib

A Telegram Library that uses Python to generate Wrappers on Any language you like :)

---

## Usage

---

## installation

```bash
pip install telelib
```

or if you're on node

```bash
npm install @TeleLib/TeleLib
# OR
yarn add @TeleLib/TeleLib
```
